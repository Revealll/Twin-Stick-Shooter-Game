# Twin-Stick-Shooter-Game
A browser-based twin-stick shooter game built with TypeScript, using object-oriented design and basic physics for gameplay

## Features
- Movement with WASD + Aiming and Shooting with mouse detection
- Zombies with basic ai chasing movement
- Collision detection
- Physics style movement (Friction and Velocity included)
- Scoring and Game-Over with restart feature

## Controls
- Move : WASD 
- Aim: Mouse
- Shoot: Left click


## How to Run
1. Install Dependancies
2. Run the index.html
